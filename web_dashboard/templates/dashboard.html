Event templates

Edit events_templates.txt here. The bot reads this file from disk on every /event create and template autocomplete — no redeploy (same server process as this dashboard). Format: each block starts with a line [Template name], then numbered lines 1. Role name. If your host resets disk on restart (e.g. some serverless setups), copy important edits into git or download a backup.

Guild role overrides

Per database guild (linked Discord server). Leave a field empty to keep using config.yaml defaults plus built-in extra IDs for that tier. Saved values replace the default set for that tier on this guild only. Clear all three fields and save to remove overrides. The bot applies changes on the next slash command (no redeploy).

Use Discord role IDs (Developer Mode → right-click role → Copy ID). Separate with commas or spaces. If discord_id for a guild is still 0, link the server first — overrides match by Discord server ID.
